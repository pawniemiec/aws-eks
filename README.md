# AWS EKS infrastructure
This repository allows you to  setup AWS Kubernetes service 


### Prerequisites
You need the following environment variables set:
```bash
AWS_DEFAULT_REGION=...
AWS_ACCESS_KEY_ID=...
AWS_SECRET_ACCESS_KEY=...
```

You need to have the following programs installed:
- terraform >= v0.11.10
